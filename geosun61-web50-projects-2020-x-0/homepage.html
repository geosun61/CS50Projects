Capstone Project

My final capstone project at Humber College that I worked on collaboratively was an IoT parking lot system using a Raspberry Pi, Android App and Firebase Database. The Android App manages the parking tickets and shows a map of the parking lot of which parking spots are available to reserve. The Raspberry Pi controls the four sensors/effectors for the project.

4 Sensors/Effectors used for project:
  • IR Break Beam
  • VCNL4010 Proximity Sensor
  • Servo-Motor
  • HD Camera
